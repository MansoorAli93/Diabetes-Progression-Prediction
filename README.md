# Diabetes Progression Regression 

## Overview
This project is a machine learning web application that predicts diabetes progression using Linear Regression and visualizes model performance with interactive charts.

## Features
- Linear Regression model training
- Mean Squared Error and RÂ² score display
- True vs Predicted value visualization
- Feature vs Prediction scatter plot
- Simple Streamlit web interface

## Technologies Used
- Python
- Streamlit
- Scikit-learn
- NumPy
- Matplotlib

## How to Run
Install dependencies:
pip install -r requirements.txt

Run the app:
streamlit run app.py
